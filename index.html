How it works: After connecting, each task card shows two Calendar buttons — Deadline (pushes the deadline date as one event) and Work Days (pushes each scheduled work day as a separate all-day or timed event). Both use duplicate checking so no repeats are created. Editing a task and re-syncing will update the existing deadline event.
Token expiry: Google tokens expire after ~1 hour. If Calendar sync stops working, click Reconnect and re-authorise. This is a Google security rule and cannot be changed in a browser-only app.

📅 Google Calendar Sync
Each scheduled work day creates a Google Calendar event at your set start time. Connect Calendar in Configure → Google Calendar, then click Add to Calendar on task cards.
Connect Google Calendar
✓ The Google Calendar API is completely free — no subscription needed. You only need a free Google account.
One-time setup (about 5 minutes):
  1. Go to console.cloud.google.com and sign in. Click New Project, give it any name, click Create.
  2. In the search bar type Google Calendar API, click it, then click Enable.
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID. Choose Web application. Under Authorised JavaScript origins add your hosted URL (e.g. https://yourusername.github.io). Click Create.
  4. Copy the Client ID shown (ends in .apps.googleusercontent.com) and paste it below.
